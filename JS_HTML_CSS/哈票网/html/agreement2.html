网站服务协议

(1)哈票网的性质
      哈票网仅为销售服务提供者，而非银行或任何其他形式的金融机构。

(2)会员注册
      用户可以通过以下两种途径注册为哈票网会员：
      
（一）线上注册
      登陆哈票网，按照哈票网上指定的流程和要求申请注册为哈票网会员，并获得一个专属于用户自己的会员账户。
（二）线下申请
       用户在哈票网合作的影院或哈票网指定的其他地点办理并领取哈票网会员卡即成为哈票网会员。哈票网为会员办理会员卡，将收取手续费人民币20元整，该手续费在会员办理会员卡退卡时不予退还。哈票网可以根据实际情况调整手续费收费标准。

(3)会员账户/会员卡充值
      4.1充值方式
       会员可以通过以下两种途径对会员账户及会员卡进行充值：
       （一）线上充值
       在哈票网上通过哈票网提供的充值平台充值。
      （二）线下充值
       在哈票网合作影院或者哈票网指定的其他地点使用现金或哈票网提供的其他支付工具充值。

(4)退票和换票
      您在哈票网购买的电影票及其他产品概不退换。一旦您在哈票网选座并确定支付后，即视为您已完成电影票的购买，您不能再主张退票或换票，但因哈票网和影院的原因导致电影确实无法观看的除外。

(5)费用与收费
      本公司的费用及收费，可在哈票网查阅。

(6)法律适用和争议解决
      本协议适用中华人民共和国法律。因本协议引致的任何纠纷，用户和本公司均应友好协商解决，协商不成的，应提交本公司住所地有管辖权的人民法院通过诉讼解决
